# Bot Events Fama ДЛЯ Majestic — Discord бот для мероприятий

Бот с системой регистрации на мероприятия: основной состав, кнопки управления и выпадающее меню.

**Разработчик:** [SaHeR](https://github.com/saher228)  
- GitHub: https://github.com/saher228  
- Сайт: https://saher.inzeworld.com/

## Установка

```bash
npm install
```

## Настройка .env

Переименуйте `.env.example` в `.env` и заполните:
**Discord Portal:** [Devolpers](https://discord.com/developers/applications)  

| Переменная | Описание |
|------------|----------|
| `DISCORD_BOT_TOKEN` | Токен бота |
| `DISCORD_CLIENT_ID` | Application ID бота |
| `DISCORD_GUILD_ID` | ID сервера для регистрации команд |
| `DISCORD_ADMIN_USER_IDS` | ID пользователей-админов через запятую |
| `DISCORD_ADMIN_ROLE_IDS` | ID ролей администраторов |
| `DISCORD_EVENT_PING_ROLE_ID` | ID роли для пинга при создании и начале ивента |
| `BOT_STATUS_ACTIVITY` | Текст статуса «Смотрит:» |
| `EVENT_GRATITUDE_MESSAGE` | Благодарность после завершения |
| `EVENTS_STORAGE_PATH` | Путь к JSON |
| `EMBED_COLOR_*` | Цвета embed (hex без #) |

## Запуск

```bash
npm run dev
```

или после сборки:

```bash
npm run build

```

## Команды

- `/event create` — создать мероприятие (только админы)
- `/event delete id:<ID>` — удалить мероприятие (только создатель)

## Создание мероприятия

При создании заполняются поля:
- **Название**
- **Сервер, Место** (напр. `Phoenix | СКЛАД #6`)
- **Время, Лимит** (напр. `17:20 | 35`)
- **Код группы, Карта** (напр. `ABC-123 | Карта`)
- **Цвет Формы итд** (Что угодно)

Пустые поля не отображаются в ивенте.

## Кнопки на сообщении мероприятия

- **Откинуть +** — записаться в состав
- **Отменить запись** — выйти из состава
- **Завершить** — завершить мероприятие (только админы)

## Выпадающее меню (только для админов)

- **Перенести** — изменить время (только админы)
- **Исключить участника из списка** — убрать участника (только админы)
- **Настроить ивент** — изменить все параметры (только админы)

## Автоматический пинг

Когда наступает время мероприятия (по МСК), бот отправляет:
```
@Роль ВСЕ КТО В СПИСКЕ СТАКАЕМСЯ НА [МЕСТО]
Код группы: [КОД]
```
